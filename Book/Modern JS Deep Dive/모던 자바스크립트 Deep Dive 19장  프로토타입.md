제대로 블로그를 시작한 지, 한 달째 이제 자바스크립트를 배운다.

정리글에 쓸모없는 내용이 적혀서 미안함을 보내지만, 두근거림이 너무 커서 이를 적고 싶었다.



자바스크립트는 명령형, 함수형, 프로토타입 기반 객체지향 프로그래밍을 지원하는 멀티 패러다임 프로그래밍 언어이다.

이 중 객체지향 프로그래밍 언어인 인유를 살펴본다.



### **객체지향 프로그래밍** 

객체지향 프로그래밍은 전통적인 명령형 프로그래밍을 벗어나 객체의 집합으로 프로그램을 표현하는 프로그래밍 패러다임을 말한다. 

실체(현실에 존재하는 특정 대상)는 특징이나 성질을 나타내는 속성을 가지고 있고, 이를 통해서 실체를 구별할 수 있다.

**이 중 우리가 구현하려는 프로그램에 필요한 속성만을 간추려 내어 표현하는 것을 추상화라 한다.**

﻿

```
const person = {
    name : 'lee',
    address: 'seoul'
    speak() {
    	console.log(name)
        console.log(address)
    }
}
```

실제로, 사람은 이름과 주소 말고도 나이, 직장 등등 다양한 속성이 있지만 이렇게 추상화할 수 있다. 이처럼 속성을 통해 **여러 개의 값을 하나의 단위로 구성한 복합적인 자료구조를 객체**라 한다.

또한, 이름과 주소는 객체의 상태를 나타내는 데이터이며, 이름과 주소를 말하는 동작도 존재한다.

즉, **객체는 상태 데이터와 동작을 하나의 논리적인 단위로 묶은 복합 자료구조**이다.
