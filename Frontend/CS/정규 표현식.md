# 정규 표현식

- 정의 : 정규 표현식, 또는 정규식은 문자열에서 특정 문자 조합을 찾기 위한 패턴입니다. 



### 정규표현식 만드는 법

- 정규 표현식 리터럴: 다음과 같이 슬래시로 패턴을 감싸서 작성합니다

```
const re = /ab+c/
```

- RegExp 객체의 생성자 호출

```
const re = new RegExp('ab+c')
```
