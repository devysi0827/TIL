# Readme

### Goal

- 알고리즘의 이해 및 실력 상승

- FE 로드맵 관련 모든 CS 지식 습득

- 사이드 프로젝트를 통한 운영 경험 및 개발 경험 습득

- 개발 트렌드 파악 및 학습하기

### Commit Rule

- 해당 커밋룰을 따라감. ex). [algo] leetcode_12

- 중복되는 prefix가 있을 시, 위에 적힌 분류체계를 따라감

  - algo : 알고리즘 관련

  - cs : 컴퓨터 과학 관련

  - fe : 프론트엔드 관련 기술스택 또는 세미나 내용

  - be : 백엔드 관련 기술스택 또는 세미나 내용

  - chore : 사소한 수정

  - etc : 위에 안 속하는 모든 것
