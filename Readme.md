### Commit Rule

- `[perfix] 내용` 형식을 따라감 ex). [algo] leetcode_12

- 중복되는 prefix가 있을 시, 상위 분류체계를 사용함

  - algo : 알고리즘 관련
  - cs : 컴퓨터 과학 관련
  - fe : 프론트엔드 관련 기술스택 또는 세미나 내용
  - be : 백엔드 관련 기술스택 또는 세미나 내용
  - project : project 하면서 배운 것들
  - chore : 가벼운 수정
  - etc : 위에 속하지 않는 모든 것
