# HTTP 에러 업데이트


Response| Explain | Note 
:-:|:-:|:-:
200|성공|  
400|Bad Request| 에러의 원인이 주로 프론트에 있으며, 원인을 찾아봐야함. 
401|권한 없음| 접근 또는 허용되지 않는 권한 
403|금지| 해당 페이지에 접근은 금지 되어 있음 
404|Not Found| 요청한 문서를 찾지 못함, 가끔 캐시가 남아있어서 그럴 때가 있음 
405|Method Not Allowed| 허용되지 않은 메서드 
500|잘못된 접근| 에러의 원인이 주로 백엔드에 있으며, 통신문제일 가능성도 있음
