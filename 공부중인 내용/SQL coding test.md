# SQL coding test

### 문법

SELECT : 

FROM : 테이블 위치 지정

ORDER BY : 정순으로 정렬, 두 개 이상일 경우 차례대로 정렬, `DESC` 역순으로 정렬

YEAR, MONTH, DAY : 2021-01-10 에서 각각 년,월,일을 가져옴

COUNT : 숫자를 세는 용도

LIMIT : 상위 N개 까지로 제한

WHERE : 구체적인 조건 `AND` 와 `OR`로 조건을 추가할 수 있다.

GROUP BY : 그루핑

HAVING : 그루핑에 해당하는 조건 (`WHERE`은 그루핑에 해당하는 조건을 사용할 수 없다)
