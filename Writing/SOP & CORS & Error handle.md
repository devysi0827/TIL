# CORS 

### CORS(Cross-Origin Resource Sharing)란?

한국어로 직역하면 "교차 출처 자원 공유"입니다.

즉, HTTP에서 다른 출처의 자원을 사용할 수 있도록 허용하는 규정입니다.

이렇게 다른 출처의 자원을 사용할 수 없는 이유는 SOP 때문입니다.



### SOP(Same-origin policy)란?

"동일 출처 정책"입니다.
