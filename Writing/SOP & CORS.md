# SOP 와 CORS  

웹에서 흔히 CORS 에러를 마주치고는 한다. 

이 에러가 왜 발생하는 지 이유를 알아보고, 해결책을 찾아보자



### SOP(Same-origin policy)란?

"동일 출처 정책"으로 즉, 같은 출처(Origin)에서만 리소스를 공유할 수 있다는 정책이다.

이런 규제가 생긴 이유는 CSRF(Cross-Site Request Forgery)나 XSS(Cross-Site Scripting) 웹 보안과 관련된 



### CORS(Cross-Origin Resource Sharing)란?

한국어로 직역하면 "교차 출처 자원 공유"입니다.

즉, 브라우저에서 다른 출처의 자원을 사용할 수 있도록 허용하는 규정입니다.

브라우저는 원칙적으로 다른 출처의 자원을 사용할 수 없습니다.



### SOP(Same-origin policy)란?

"동일 출처 정책"입니다.
